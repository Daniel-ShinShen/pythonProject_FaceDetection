# People_Counting_Yolov8+Deepsort

## Project introduction:
基於魚眼鏡頭與監視器影像下人流統計之需求，分別使用Yolov8與DeepSort於錄像中進行行人偵測、追蹤與計數，搭配使用 pyQt5 設計 UI 介面，以便掌握不同時段特定區域的人數變化，協助管理者掌握現場狀況。


## Process flow:
1. Data Collection
2. Object Detection
3. Object Tracking
4. People Counting
